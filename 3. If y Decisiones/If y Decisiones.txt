---------------------------------------------------------------------------------
¿Qué es IF?
---------------------------------------------------------------------------------
IF significa:

SI

Sirve para decir:

SI pasa algo
ENTONCES haz esto

Ejemplo humano:
SI llueve
ENTONCES usa sombrilla




---------------------------------------------------------------------------------
Sintaxis básica
---------------------------------------------------------------------------------

IF condicion THEN
    instrucciones
END IF




---------------------------------------------------------------------------------
Primer ejemplo
---------------------------------------------------------------------------------

edad = 18

IF edad >= 18 THEN
    PRINT "Eres mayor de edad"
END IF




---------------------------------------------------------------------------------
Qué significa >=
---------------------------------------------------------------------------------
## Operadores de comparación


| Símbolo | Significa   |
|   ---   |    ---      |
|    =    |   igual     |
|    >    |   mayor     | 
|    <    |   menor     |
|    >=   | mayor o igual |
|    <=   | menor o igual |
|    <>   |  diferente  |




---------------------------------------------------------------------------------
Ejemplos rápidos
---------------------------------------------------------------------------------
IF 5 > 2 THEN
    PRINT "Correcto"
END IF


IF 10 = 10 THEN
    PRINT "Son iguales"
END IF


IF 7 <> 3 THEN
    PRINT "Son diferentes"
END IF




---------------------------------------------------------------------------------
INPUT + IF
---------------------------------------------------------------------------------
Aquí ya hacemos programas interactivos.

INPUT "Tu edad: ", edad
IF edad >= 18 THEN
    PRINT "Puedes entrar"
END IF




---------------------------------------------------------------------------------
ELSE — si NO pasa
---------------------------------------------------------------------------------
`ELSE` significa:

SI NO

Ejemplo:
INPUT "Tu edad: ", edad

IF edad >= 18 THEN
    PRINT "Mayor de edad"
ELSE
    PRINT "Menor de edad"
END IF




---------------------------------------------------------------------------------
Cómo piensa BASIC
---------------------------------------------------------------------------------
Si escribes:

17

el programa piensa:

17 >= 18


Eso es falso.

Entonces ejecuta:

ELSE




---------------------------------------------------------------------------------
Programas con decisiones
---------------------------------------------------------------------------------
## Ejemplo — contraseña

INPUT "Clave: ", clave$

IF clave$ = "1234" THEN
    PRINT "Acceso permitido"
ELSE
    PRINT "Clave incorrecta"
END IF




---------------------------------------------------------------------------------
Comparar texto
---------------------------------------------------------------------------------
También puedes comparar palabras.

Ejemplo:
INPUT "Color favorito: ", color$

IF color$ = "azul" THEN
    PRINT "Buen gusto"
END IF




---------------------------------------------------------------------------------
IF anidados
---------------------------------------------------------------------------------
Un IF dentro de otro.

edad = 20

IF edad >= 18 THEN

    IF edad >= 60 THEN
        PRINT "Adulto mayor"
    ELSE
        PRINT "Adulto"
    END IF

END IF




---------------------------------------------------------------------------------
ELSEIF — varias opciones
---------------------------------------------------------------------------------

nota = 85

IF nota >= 90 THEN
    PRINT "Excelente"

ELSEIF nota >= 70 THEN
    PRINT "Aprobado"

ELSE
    PRINT "Reprobado"
END IF




---------------------------------------------------------------------------------
Lógica booleana
---------------------------------------------------------------------------------
Las condiciones solo pueden ser:

VERDADERO

o

FALSO

No hay punto medio.




---------------------------------------------------------------------------------
Operadores lógicos
---------------------------------------------------------------------------------


| Operador | Significa |
|   ---    |    ---    | 
|   AND    |     Y     |
|   OR     |     O     |
|   NOT    |     NO    |




---------------------------------------------------------------------------------
Ejemplo con AND
---------------------------------------------------------------------------------

edad = 20
dinero = 50

IF edad >= 18 AND dinero >= 20 THEN
    PRINT "Puedes entrar"
END IF


Las DOS condiciones deben cumplirse.




---------------------------------------------------------------------------------
Ejemplo con OR
---------------------------------------------------------------------------------

IF color$ = "rojo" OR color$ = "azul" THEN
    PRINT "Color permitido"
END IF

Solo una necesita ser verdadera.




---------------------------------------------------------------------------------
⚠ Error común
---------------------------------------------------------------------------------
MAL:

IF edad >= 18
PRINT "Hola"


Falta:

THEN
