-------------------------------------------------------------------------------
¿Qué es Basic?
-------------------------------------------------------------------------------
Siglas de Beginners All-purpose Symbolic Instruction Code 
(Código simbólico de instrucciones de propósito general para principiantes)

El BASIC original, el Dartmouth BASIC, fue diseñado en 1964 por John George 
Kemeny, Thomas Eugene Kurtz y Mary Kenneth Keller en el Dartmouth College 
en New Hampshire, Estados Unidos, como un medio para facilitar la programación 
en ordenadores a estudiantes (y profesores) que no fueran de ciencias. 
En aquella época, casi todo el uso de los computadores requería codificar 
software hecho a medida, con lo cual quedaba restringido a personas con 
formación como científicos y matemáticos.

    1. Levántate

Ejemplo en BASIC:

---------------------------------------------------------------------------------
¿Qué hace PRINT?
---------------------------------------------------------------------------------

PRINT significa:

> Mostrar texto en pantalla.



---------------------------------------------------------------------------------
Las comillas
---------------------------------------------------------------------------------
Todo texto debe ir entre comillas.

Correcto:
PRINT "Hola"


Incorrecto:
PRINT Hola

Porque BASIC pensará que `Hola` es una variable.


---------------------------------------------------------------------------------
Varias líneas
---------------------------------------------------------------------------------

Cada instrucción normalmente va en una línea distinta:

PRINT "Linea 1"
PRINT "Linea 2"
PRINT "Linea 3"



---------------------------------------------------------------------------------
CLS — limpiar pantalla
---------------------------------------------------------------------------------

CLS

Significa:

> Clear Screen

Limpia la consola.





---------------------------------------------------------------------------------
END — terminar programa
---------------------------------------------------------------------------------

END

Finaliza el programa.

Ejemplo:

PRINT "Inicio"
END
PRINT "Esto no aparece"

Tu primer mini programa
---------------------------------------------------------------------------------

PRINT "================="
